# 23281191 机器学习作业

本仓库用于存放机器学习课程的作业，主要内容为超像素相关的研究和实验。

## 仓库结构

```
23281191-/
├── assignments/     # 作业目录，每个作业一个子目录
├── data/           # 数据集目录
├── docs/           # 文档和笔记目录
├── results/        # 实验结果目录
└── README.md       # 本文件
```

## 使用说明

1. **作业存放**: 所有作业文件存放在 `assignments/` 目录下
2. **数据管理**: 实验数据集存放在 `data/` 目录下
3. **结果保存**: 实验结果和输出文件保存在 `results/` 目录下
4. **文档记录**: 相关文档和笔记保存在 `docs/` 目录下

## 课程主题

- 超像素（Superpixel）算法
- 图像分割
- 机器学习应用

## 环境配置

推荐使用 Python 3.x 环境，常用库包括：
- NumPy
- OpenCV
- scikit-image
- Matplotlib

具体依赖请参考各个作业目录中的说明。
